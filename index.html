Projects

Eye of the Tiger
Activity 1

This activity demonstrates the use of basic HTML structure and formatting by recreating the lyrics of the song "Eye of the Tiger." It focuses on organizing content using headings, paragraphs, and line breaks, helping students practice semantic markup and text presentation in web development.

Resume w/o Lists
Activity 2

This activity showcases a personal resume using only paragraphs and blockquotes, without using HTML list elements. It demonstrates how to structure and present contact information, objectives, educational background, skills, experience, projects, and certifications in a clean, readable format using semantic HTML and CSS for styling.

Resume with Lists
Activity 3

This activity presents a personal resume using semantic HTML elements, including both paragraphs and lists. It organizes contact information, objectives, educational background, skills, experience, projects, and certifications in a structured format, demonstrating how to effectively use lists for clear and accessible content presentation in web development.

Map area
Activity 4

This activity demonstrates the use of HTML image maps by creating clickable regions on a "Holiday Cakes" image. Each area links to a different page, allowing users to interact with specific parts of the image and navigate to related content. This showcases how image maps can enhance web interactivity and navigation.

Table colspan and rowspan
Activity 5

This activity demonstrates advanced HTML table structures using rowspan and colspan attributes. It features multiple tables with complex cell arrangements, helping students practice creating organized and visually appealing tabular data layouts for various use cases.

Calendar w/o div
Activity 6

This activity features a fully structured HTML calendar for the year 2025, including month navigation links and a list of holidays. It demonstrates how to organize calendar data using tables, highlight special dates, and provide quick navigation between months, helping students practice advanced table layouts and internal page linking.

Calendar with div
Activity 7

This activity presents a comprehensive 2025 calendar using HTML tables, complete with month navigation links and a grid of clickable holiday dates. It demonstrates how to combine advanced table layouts, internal linking, and grid-based holiday highlights to create an interactive and organized calendar web page.

School registration form
Activity 8

This activity features a styled online application form for the International State College of the Philippines. It collects applicant details such as campus, academic term, program, and personal information, using organized sections and clear reminders. The form demonstrates best practices in form structure, validation, and user-friendly web design.

Navigation and Animation
Activity 9

This activity demonstrates how to create a navigation bar with a dropdown menu using HTML and CSS. It allows users to access multiple pages from a single menu, showcasing interactive navigation design and enhancing user experience on the website.

UP website clone
Activity 10

This activity recreates the homepage of the University of the Philippines Diliman, featuring a video banner, overview, latest news, research and extension highlights, upcoming events, and a contact section. It demonstrates advanced HTML and CSS layout techniques, multimedia integration, and responsive design to provide a comprehensive and interactive university website experience.
